# Raw Data Directory

This directory is intended for storing raw input data files before processing. These files should be in their original format as received from the data source.

## Expected File Types
- Excel files (.xlsx, .xls)
- CSV files (.csv)
- JSON files (.json)

## Data Organization
- Keep original file names
- Do not modify the raw data files
- Document any data quality issues in a separate metadata file

## Data Privacy
- Ensure all data files contain de-identified information
- Follow data protection guidelines
- Do not commit sensitive data to version control
